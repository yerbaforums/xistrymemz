================================================================================
                      REDISTRIBUTION OPTIONS
================================================================================

1. Self-Host (Clearnet):
   docker compose up -d

2. Tor Hidden Service:
   docker compose --profile tor up -d
   # Access via http://xxx.onion

3. I2P Hidden Service:
   docker compose --profile i2p up -d
   # Access via http://localhost:4444

4. Custom Domain:
   docker run -p 3000:3000 \
     -e NEXT_PUBLIC_URL=https://your-domain.com \
     -e INSTANCE_NAME="YourCommunity" \
     earthlite

5. Federation Enabled:
   docker run -p 3000:3000 \
     -e ENABLE_FEDERATION=true \
     earthlite
